start - Start the bot
help - Show the help menu
play - Play audio from a link or search query
vplay - Play video from a link or search query
stop - Stop playback and clear queue
pause - Pause the current playback
resume - Resume paused playback
skip - Skip to the next track
queue - Show the list of queued tracks
seek - Seek forward
seekback - Seek backward
autoplay - Toggle smart autoplay mode
anime - Search and stream anime
addsession - Add a new assistant session (sudo only)
rmsession - Remove an assistant session (sudo only)
sessions - List all active assistants (sudo only)
ping - Check bot latency and status
stats - Show bot statistics
